## Structural finding (pre-scorecard)

All 5 agents in this campaign are marked "(implicit)" or "fired after" in the FINAL REPORT's
Section 2. None produced a structured verdict block. None disclosed worktree path, branch, or
SHA. This is a campaign-level structural gap: agents are being named as participants in a report
without being formally dispatched via Task tool and without returning their canonical output format.

This is distinguished from GATE 5 substitution (a missing agent replaced by an available one)
— it is a dispatch-mode failure: the orchestrator attributed work to registered agent names but
the work was performed inline without formal agent invocation. Consequence: the 7-dimension
scoring rubric has less to work with than a properly-dispatched campaign; scores on Substitution,
Evidence, and Environment will be systematically penalised across all 5 agents.

This structural finding is itself a GATE 4 salvage item: "implicit agent attribution" is a
pattern that degrades scorecard reliability. See Section 3 (Weak-verdict warnings) for
recommended disposition.
